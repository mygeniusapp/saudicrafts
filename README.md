# SaudiCrafts - Saudi Handmade Marketplace

SaudiCrafts is an online marketplace connecting local Saudi artisans with customers who value authentic handmade crafts. The platform supports bilingual (Arabic/English) browsing, product filtering, and user authentication (demo).

## Features
- Responsive design with Bootstrap 5
- Bilingual interface (Arabic/English toggle)
- Product filtering and sorting
- Shopping cart simulation
- User login/registration with client-side validation
- About Us and Join Us pages with Vision 2030 alignment

## Technologies
- HTML5, CSS3, JavaScript
- Bootstrap 5, Font Awesome
- jQuery (for Bootstrap components)
